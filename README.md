## Описание
[⬆️ Наверх](#содержание)

**Steam Deck Helper 2.0** — средство для упрощения работы с префиксами и играми на Steam Deck: установка библиотек, очистка префиксов и кэша шейдеров, автоматизация установки программ, применение оптимизаций и многое другое.

---

## Установка
[⬆️ Наверх](#содержание)

1. **Скачайте инсталлятор** и положите его на Рабочий стол:
   - `[steam_deck_helper_installer.desktop](https://github.com/allians00758/steam_deck_helper/releases/download/dlls/steam_deck_helper_installer.desktop)
   - Загрузки доступны в нашем репозитории.

2. **Запустите** файл, нажмите **Продолжить** и следуйте подсказкам.

3. Если у вас **не задан пароль sudo**, инсталлятор предложит его установить (окно откроет Konsole и даст инструкции).

> ⚠️ При вводе пароля символы **не отображаются** — это нормально.

4. После установки на Рабочем столе появится ярлык **Steam Deck Helper**. Запустите программу.

### Если нет браузера (через Konsole)

Откройте **Konsole** (поиск в меню Пуск) и выполните:

```bash
curl -L -O https://github.com/allians00758/steam_deck_helper/releases/download/dlls/steam_deck_helper_installer.desktop
```
или

```bash
curl -L -O https://repo.steamdeckbase.org/repo/main/repository/sdh_install/steam_deck_helper_installer.desktop
```

Переместите файл на Рабочий стол и запустите двойным кликом.

![Главное окно](https://img.steamdeckbase.org/PtjW34rutaN2321/images/68c874d9c4b0b.png)

### Первый запуск и первичная настройка

1. Выбор языка приложения.
2. Предложение **включить автоматические фоновые обновления** при запуске.
3. Уведомления об обновлениях появляются в правой части экрана.
4. Программа предложит **сгенерировать locale** для корректной работы системы: нажмите **Да**, введите `sudo`‑пароль и дождитесь завершения.
5. После этого откроется **основное окно** программы.

## Инструкция по использованию

### Главное меню
[⬆️ Наверх](#содержание)

![Главное окно](https://img.steamdeckbase.org/PtjW34rutaN2321/images/68e9ef72e0a6a.png)

Главное меню разделено на три части:

1. **Настройки и информация**
   - Смена языка (требуется перезапуск).
   - Включение/отключение автообновлений.
   - О проекте и создателях.
   - Отправка лога работы (при ошибках).
   - Список изменений (обновляемый).
   - Справка (эта инструкция).
2. **Блок ссылок**
   - Telegram‑группы и YouTube‑канал.
   - Страница с реквизитами («Сказать спасибо»).
   - Мини‑игра «Сапёр».
3. **Основное меню**
   - **Менеджер префиксов**.
   - **Параметры запуска** (применение параметров к ярлыкам Steam).
   - **Инструменты** (установка утилит, программ и исправлений).
   - **Deck‑оптимизации**.
   - **Выход**.

### Инструменты
[⬆️ Наверх](#содержание)

![Инструменты-меню](https://i.ibb.co/HpL6svks/image.png)

Раздел для автоматизированной установки программ, утилит и исправлений. Доступно (состав может меняться):

1. **Загрузка с рабочего стола** — переключение загрузки устройства между рабочим и игровым режимом.
2. **CDEmu (SteamOS 3.7)** — монтирование образов разных форматов.
3. **Изменение TTL (ttl=65)** — меняет TTL для разблокировки ограничений мобильного оператора на скорость загрузки.
4. **Decky Loader** — установка/обновление магазина плагинов, удаление плагинов.
5. **Исправление Discover** — исправляет установку приложений из Discover.
6. **Отключение KDE Wallet** — отключение надоедливого окна запрашивающего пароль (инструмент для управления паролями в системе KDE Plasma).
7. **DXVK Low Latency (latest)** — заменяет dll файлы DXVK на последнюю актуальную версию в кастомных версиях протонов (GE, cachyos, proton-tkg и другие сборки) на Ваш выбор. Служит для снижения задержек рендера и более плавного управления (Несовместим с sd-helper-optiscaler из Upscalers Mods).
   `DXVK_CONFIG_FILE=~/.config/steam_deck_helper/dxvk.conf %command%`  
   > Несовместимо с модом **sd-helper-optiscaler** из раздела «Upscalers Mods».
8. **EmuDeck** — хаб с готовыми настройками для множества эмуляторов.
9. **Исправление для Forza Horizon 4** — фикс для вылетов в осенних гонках.
10. **Исправление Hogwarts Legacy** — фикс запуска игры.
11. **Lampa + VLC** — установка онлайн‑кинотеатра.
12. **Исправление локалей и раскладки** (игровой режим, сейчас русская):
    - *Russian Locales Fix* — убирает «крокозябры» и включает корректный ru_RU.UTF‑8 без `HOST_LC_ALL=ru_RU.UTF-8 %command%`.
    - *Russian Keyboard Fix (Alt+Shift)* — переключение раскладки в игровом режиме с помощью внешней клоавиатуры (ставьте только при использовании внешней клавиатуры, добавляет баги в цифровую при написании символов).
13. **Лаунчеры** — Epic Games, Battle.net, VK Play, Ubisoft Connect (ярлыки добавляются в Steam; путь установки: `/home/deck/Games`).
14. **Сброс звуковых настроек по умолчанию** - сбрасывает настройки звука системы на дефолтные значения, может помочь если у вас пропал звук в системе, например после использования Bluetooth наушников.
15. **Рекомендуемые программы** — быстрая установка приложений из Discover.
16. **ReShade (latest)** — установка Reshader и автоматизированное добавление в игры через диалоговые окна.
17. **Восстановить ярлык Игрового Режима** — восстанавливает ярлык на рабочем столе перехода в игровой режим.
18. **SMB Mount** — утилита для монтирования сетевых папок. 
19. **Включение SSH** — активация SSH‑доступа.
20. **Исправление эмулятора Xenia** - решает проблему с запуском некоторых игр.
21. **Исправление The Crew** - решает проблему с запуском игр.
22. **Upscalers Mods** — мод **sd-helper-optiscaler** (генерация кадров, подмена DLSS на FSR 3.1.5).  
    > Несовместим с **DXVK Low Latency (latest)**.
23. **yay** - помощник pacman с поддержкой репозитория aur. 
24. **ZeroTier One** — виртуальная локальная сеть.

#### Установка библиотек и настройка префикса
[⬆️ Наверх](#содержание)

![https://i.ibb.co/HLfjW1mp/image.png](https://i.ibb.co/HLfjW1mp/image.png)
![https://i.ibb.co/fY1NzR5B/image.png](https://i.ibb.co/fY1NzR5B/image.png)

Доступно после выбора префикса.

Функции:
1. **Установить выбранный набор библиотек** — тихая установка заранее собранных наборов.
2. **Свои библиотеки** — быстрый доступ к установке **вашего** набора для выбранного префикса (тихий режим).
3. **Конфигурация Wine** — настройка параметров префикса.
4. **Установка/Удаление программ** — быстрый доступ к соответствующему инструменту для выбранного префикса.
5. **Настройки префикса** — смена настроек префикса.
6. **Реестр** — доступ к реестру выбранного префикса.

> Наборы библиотек подготовлены для всех игр, разобранных в группе телеграмм **Steam Deck База Знаний**.
